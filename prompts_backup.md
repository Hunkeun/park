# NotebookLM 생성 프롬프트 백업
이 파일은 오늘 생성한 4개의 슬라이드 세트와 4개의 오디오 오버뷰를 만들기 위해 사용되었던 **맞춤형 명령어(Custom Instructions) 원문**입니다.
후에 비슷한 형식의 자료를 생성할 때 복사하여 재사용할 수 있습니다.

---

## 📑 1. 슬라이드 덱 (Slide Decks) 프롬프트

### 세트 1/4 (The MCP Standard)
> 세트 1/4: MCP(Model Context Protocol)의 기본 개념과 원리에 집중하세요. MCP란 무엇인지, MCP의 탄생 배경, MCP의 핵심 아키텍처(호스트-클라이언트-서버 구조), MCP의 동작 원리, MCP가 AI 도구 통합에서 왜 중요한지, MCP의 보안 고려사항을 다루세요. NotebookLM 설치나 Antigravity 연동은 다루지 마세요. 이 부분은 이후 슬라이드에서 다룹니다.

### 세트 2/4 (Windows NotebookLM MCP Pipeline)
> 세트 2/4: NotebookLM MCP 서버의 설치와 설정 과정에 집중하세요. Python 설치, pip를 통한 notebooklm-mcp 패키지 설치, nlm login 명령어를 통한 구글 계정 인증, 쿠키 기반 세션 관리, mcp_config.json 설정 파일 작성법, 환경 변수 설정, MCP 서버 시작과 연결 확인을 다루세요. Windows 환경 기준으로 설명하세요. MCP 기본 개념(세트1에서 다룸)이나 Antigravity 연동(세트3에서 다룸)은 다루지 마세요.

### 세트 3/4 (Antigravity NotebookLM Command Center)
> 세트 3/4: Antigravity IDE와 NotebookLM MCP의 연동에 집중하세요. Antigravity가 무엇인지, Antigravity에서 MCP 서버를 설정하는 방법, mcp_config.json에서 Antigravity 전용 구성, 소스 추가 및 노트북 관리를 Antigravity 내에서 수행하는 방법, 바이브 코딩을 통한 챗봇 구현 사례, macOS/Windows 환경 차이와 트러블슈팅을 다루세요. MCP 기본 개념(세트1)이나 설치 절차(세트2)는 다루지 마세요.

### 세트 4/4 (The MCP Knowledge Factory)
> 세트 4/4: 실전 활용 사례와 자동화 워크플로우에 집중하세요. NotebookLM MCP를 활용한 자동화 연구 워크플로우, 팟캐스트·마인드맵·퀴즈 등 콘텐츠 자동 생성, 노션 MCP와의 결합을 통한 업무 자동화, 책 쓰기와 마케팅 자동화 사례, 교차 노트북 쿼리를 통한 지식 관리, 미래 전망과 생태계 확장을 다루세요. MCP 기본 개념(세트1), 설치(세트2), Antigravity 연동(세트3)은 다루지 마세요.

## 🎙️ 2. 오디오 오버뷰 (Audio Overviews) 프롬프트

### 오디오 1/4 (AI를 위한 만능 포트 앤스로픽 MCP)
> 오디오 1/4: MCP(Model Context Protocol)의 기본 개념과 원리에 대해서만 설명하세요. MCP란 무엇인지, 탄생 배경, 호스트-클라이언트-서버 아키텍처, 동작 원리, AI 도구 통합에서의 중요성을 다루세요.

### 오디오 2/4 (윈도우 NotebookLM MCP 서버 설치와 설정)
> 오디오 2/4: NotebookLM MCP 서버의 설치와 설정 과정만 설명하세요. Python 설치, pip 패키지 설치, nlm login 인증, mcp_config.json 설정, Windows 환경 기준 절차를 다루세요.

### 오디오 3/4 (MCP로 안티그래비티와 NotebookLM 연동하기)
> 오디오 3/4: Antigravity IDE와 NotebookLM MCP의 연동에 대해서만 설명하세요. Antigravity에서 MCP 서버 설정, 소스 관리, 바이브 코딩 활용법, 트러블슈팅을 다루세요.

### 오디오 4/4 (NotebookLM과 MCP로 50페이지 책 자동 집필)
> 오디오 4/4: 실전 활용 사례와 자동화 워크플로우만 설명하세요. 팟캐스트·콘텐츠 자동 생성, 노션 연동, 책 쓰기·마케팅 자동화, 교차 노트북 쿼리, 미래 전망을 다루세요.
